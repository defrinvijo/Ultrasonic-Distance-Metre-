OVERVIEW:

This project is an Arduino-based Ultrasonic Distance Meter that measures the distance between the sensor and an object using the HC-SR04 Ultrasonic Sensor. The measured distance is displayed on a 16×2 I2C LCD in both centimeters and inches. The system also provides visual and audible alerts using LEDs and a buzzer based on the proximity of detected objects.

FEATURES:

Real time distance measurement using HC-SR04
Distance display in cm and inches
16×2 I2C LCD output
Green, Yellow, and Red LED status indicators
Buzzer warning system
Safe, Warning, and Danger zones
Serial Monitor support for debugging and monitoring
Low-cost and easy-to-build design
Expandable for IoT and smart automation applications
